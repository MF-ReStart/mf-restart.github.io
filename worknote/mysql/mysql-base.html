1. 基础

MySQL 数据库属于关系型数据库（RDBMS）。

关系型数据库的概念：建立在关系模型基础上，由多张相互连接的二维表组成的数据库。

我们通过客户端连接到 MySQL 数据库管理系统（DBMS），使用 SQL 语句通过 MySQL 数据库管理系统来创建数据库，以及创建数据库下的表。

1.1. SQL 通用语法

  • SQL 语句可以单行或多行书写，以分号结尾。
  • SQL 语句可以使用空格或缩进来增强语句的可读性。
  • MySQL 数据库的 SQL 语句不区分大小写，关键字建议使用大写。
  • 单行注释使用 --注释内容 或 #注释内容 （MySQL 特有），多行注释使用 /*注释内容*/。

1.2. SQL 分类

1.2.1. DDL （Data Definition Language）

数据定义语言，用来定义数据库对象，例如数据库、表、字段。

1.2.2. DML （Data Manipulation Language）

数据操作语言，用来对数据库表中的数据进行增删改。

1.2.3. DQL （Data Query Language）

数据查询语言，用来查询数据库中表的记录。

1.2.4. DCL （Data Control Language）

数据控制语言，用来创建数据库用户、控制数据库的访问权限。

1.3. DDL 数据库操作

1.3.1. 查询

查看所有数据库。

show databases;

查询当前所在数据库。

select database();

1.3.2. 创建

创建数据库。

create database if not exists test_db default character set utf8mb4 default collate utf8mb4_general_ci;

1.3.3. 删除

删除数据库。

drop database if exists test_db;

1.3.4. 使用

切换数据库。

use test_db;

1.4. DDL 表操作

1.4.1. 查询

查询当前数据库所有表。

show tables;

查询表结构。

desc tablename;

查询建表语句。

show create table user;
